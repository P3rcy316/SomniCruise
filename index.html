Welcome to SomniCruise!
SomniCruise is a website dedicated to those that like night drives with music. You can use the site to study, sleep, relax, game, or anything else!
This site is currently in active development and will vastly change over the course of time so please take that into account. This site is open source and the link to the github is in the bottom right.
If you have any ideas please let me know at "somnicruise.dev@gmail.com" and thank you for using the site!
